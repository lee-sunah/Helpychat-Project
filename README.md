# 🧠 AI HelpyChat QA Project!

> HelpyChat QA 프로젝트는 HelpyChat 서비스의 **핵심 기능 품질을 보증(QA)** 하고  
> **테스트 자동화 기반 CI/CD 프로세스 구축**을 목표로 합니다.  
> 본 프로젝트는 HelpyChat의 다양한 기능 —  
> **계정/조직 관리, 빌링, 채팅, 파일 분석, AI 생성 기능, 커스텀 에이전트 등**을 대상으로  
> 테스트 케이스(TC)를 설계하고, 자동화 스크립트(Pytest + Selenium 등)로 검증을 수행합니다.

## 🧩 테스트 대상 핵심 기능

### 1️⃣ 계정 / 조직
- 회원가입, 로그인, 로그아웃

### 2️⃣ 빌링 & 이용내역 
- 크레딧 잔량 표시

### 3️⃣ 채팅 히스토리 
- 새 채팅 시작 / 히스토리 검색 / 에이전트 탐색
- 대화 목록 시간순 정렬 및 제목 수정
- 과거 대화 전체 내용 조회

### 4️⃣ 채팅 기본 기능 
- AI 실시간 대화 (텍스트 기반 질의응답)
- 멀티모달 AI: 이미지·문서 등 복합 입력 처리
- 응답 복사 / 좋아요·싫어요 피드백 / 메시지 수정
- 최신 메시지로 자동 스크롤 이동

### 5️⃣ 채팅 고급 기능 
- 파일업로드: 문서, 이미지 업로드 후 AI 분석 수행 
- 심층조사: 특정 주제에 대한 심화 리서치 
- 구글 검색:  실시간 웹 검색 기반 최신 정보 활용
- PPT 제작: 주제·조건 입력 → 자동 슬라이드 생성
- 퀴즈 생성: 난이도/유형별 퀴즈 생성 (객관식·주관식) 
- 이미지 생성: 프로필, 아이콘, 시각화 이미지 생성 및 편집

### 6️⃣ 맞춤화 기능 
- 커스텀 에이전트 생성/조회/수정/삭제  
- 에이전트 설정 항목: 이름, 설명, 이미지, 규칙, 시작 대화, 기능 선택(Web Search / Code Execution 등)
- 공개 범위 설정: 전체기관공개 / 기관공개 / 나만보기 / 기본에이전트

## 🔥 트러블슈팅 핵심 요약

### ⚡ 병렬 실행 최적화
- 문제: `pytest -n 2` 실행 시 WebDriver 충돌  
- 원인: fixture가 세션을 공유  
- 해결: **function scope fixture 적용**  
- 성과: 전체 실행 속도 약 **40~50% 단축**

### ⚡ 테스트 불안정성 → Rerun 적용
- 느린 응답/랜덤 지연 → 테스트 Fail  
- `--reruns 2 --reruns-delay 2` 적용  
- **불안정 테스트 자동 보정**

### ⚡ Jenkins venv 문제 해결
- Docker Jenkins 실행 환경에서 venv가 자동 생성되지 않음  
- 해결: Jenkinsfile 내 Python 가상환경 생성 로직 추가

### ⚡ 파일 다운로드 실패 해결
- Chrome 정책으로 다운로드 차단 문제 발생  
- 해결: preferences 수정 → 자동 다운로드 허용  
- Allure 스크린샷을 통해 실패 지점 즉시 확인 가능
