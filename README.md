# 🧠 AI HelpyChat QA Project

> **"모든 AI 업무를 하나의 플랫폼에서 해결한다."**  
> HelpyChat은 엘리스 트랙에서 개발한 **통합형 AI 솔루션**으로,  
> 채팅, PPT 제작, 코드 생성, 이미지 생성, 교육 문서 생성 등  
> 여러 AI 서비스를 하나로 통합하여 **매끄럽고 안전한 기업용 경험**을 제공합니다.  

---

## 🏢 프로젝트 개요

HelpyChat QA 프로젝트는 HelpyChat 서비스의 **핵심 기능 품질을 보증(QA)** 하고  
**테스트 자동화 기반 CI/CD 프로세스 구축**을 목표로 합니다.  

본 프로젝트는 HelpyChat의 다양한 기능 —  
**계정/조직 관리, 빌링, 채팅, 파일 분석, AI 생성 기능, 커스텀 에이전트 등**을 대상으로  
테스트 케이스(TC)를 설계하고, 자동화 스크립트(Pytest + Selenium 등)로 검증을 수행합니다.

## 🎯 프로젝트 목표

1. **효율성 향상** — QA 자동화를 통해 반복 테스트를 최소화  
2. **결함 탐지** — 서비스 전반의 기능적 결함을 조기에 발견  
3. **품질 향상** — 사용자 중심 테스트로 제품 신뢰도 강화  
4. **효율적 리포트** — 테스트 결과 자동 리포트 및 로그 관리  
5. **CI/CD 지원** — 지속적 통합/배포 환경에서 테스트 자동화 수행

## 🧩 테스트 대상 핵심 기능

### 1️⃣ 계정 / 조직
- 엘리스 SSO (OAuth 기반 통합 로그인)
- 회원가입, 로그인, 로그아웃
- 사용자 권한 계층  
  - 기관소유자: 최고 관리 권한  
  - 기관관리자: 기관 내 관리 및 설정  
  - 기관교육자: 교육 콘텐츠 운영 (일반유저 권한 동일)  
  - 일반유저: 기본 AI 채팅 기능 이용
- 구성원 초대 (기관관리자 이상 사용 가능)

---

### 2️⃣ 빌링 & 이용내역 💳
- 크레딧 시스템: AI 이용량 기반 선불 결제
- 크레딧 충전 (₩10,000 ~ ₩3,000,000)
- 자동 충전: 잔액 임계값 이하 시 자동 결제
- 결제 수단 관리: 카드 등록 및 기본 결제 설정
- 이용 내역 조회: 충전/사용 이력 및 기간별 리포트

---

### 3️⃣ 채팅 히스토리 💬
- 새 채팅 시작 / 히스토리 검색 / 에이전트 탐색
- 대화 목록 시간순 정렬 및 제목 수정
- 과거 대화 전체 내용 조회

---

### 4️⃣ 채팅 기본 기능 💬
- AI 실시간 대화 (텍스트 기반 질의응답)
- 멀티모달 AI: 이미지·문서 등 복합 입력 처리
- 응답 복사 / 좋아요·싫어요 피드백 / 메시지 수정
- 최신 메시지로 자동 스크롤 이동

---

### 5️⃣ 채팅 고급 기능 ⚙️
| 기능 | 설명 |
|------|------|
| **파일 업로드** | 문서, 이미지 업로드 후 AI 분석 수행 |
| **심층 조사** | 특정 주제에 대한 심화 리서치 |
| **구글 검색** | 실시간 웹 검색 기반 최신 정보 활용 |
| **K12 기능** | 세특, 행발, 수업지도안 등 교육 문서 자동 생성 |
| **PPT 제작** | 주제·조건 입력 → 자동 슬라이드 생성 (텍스트+차트+이미지 구성) |
| **퀴즈 생성** | 난이도/유형별 퀴즈 생성 (객관식·주관식) |
| **이미지 생성** | 프로필, 아이콘, 시각화 이미지 생성 및 편집 |
| **차트/그래프 생성** | CSV 기반 막대/파이/꺾은선 차트 자동 시각화 |

---

### 6️⃣ 맞춤화 기능 🔎
- 커스텀 에이전트 생성/조회/수정/삭제  
- 에이전트 설정 항목  
  - 이름, 설명, 이미지, 규칙, 시작 대화, 기능 선택(Web Search / Code Execution 등)  
- 공개 범위 설정  
  - 전체기관공개 / 기관공개 / 나만보기 / 기본에이전트

## 🧪 테스트 설계 개요

| 구분 | 검증 범위 | 목적 |
|------|------------|------|
| **기능 테스트** | 각 기능의 정상 동작 (Happy Path) | 기능 정확성 확인 |
| **예외 테스트** | 입력 누락, 비정상 값, 제한 초과 | 안정성 검증 |
| **UI 테스트** | 버튼/메뉴/피드백 메시지 | 사용성 확인 |
| **보안 테스트** | 인증, 접근권한, 필터링 | 안전성 검증 |
| **성능 테스트** | 대용량 파일, 응답속도 | 효율성 검증 |
| **자동화 테스트** | 반복 시나리오 자동 수행 | 효율화 및 CI/CD 연동 |
